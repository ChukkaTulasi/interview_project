Aptitude Questions

Q: What is the average of first 10 natural numbers?

A: 5.5

Q: Time and Work: A can do a work in 10 days, B in 20 days. How long together?

A: 6.67 days

Technical Questions

Q: What is a pointer in C?

A: A pointer is a variable that stores the address of another variable.

Q: Difference between Java and JavaScript?

A: Java is a programming language; JavaScript is a scripting language for web pages.

HR Questions

A: [Insert your personal intro with skills, goals, and achievements]

Resume Tips

  • Use a clean, professional layout.
  • Highlight your projects and skills.
  • Use action words like “Developed”, “Built”, “Led”.
  • Keep it under 1-2 pages.
